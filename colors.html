Background colors.

HEX CODE NAME USE CASE DESCRIPTION
#000000
 BLACK This color should only be used as text color for text on a dark background or as an outline color/accent color.
#111111
 DEEP DARK This color should be used as a general accent color when marking something as being different/less important than the main UI.
#1e1e1e
 BASE DARK This color should be used as the main background color for UI elements.
#2b2b2b
 LIGHT DARK This color should be used as a transition color or for signalling that a UI element is not in focus.
#353535
 BASE GREY This color should be used as background color for accented or important UI elements.
